# What's new in 11.3

## .NET 8 support for SOEs and SOIs

Starting from ArcGIS Enterprise SDK 11.3, developing .NET-based SOEs and SOIs with .NET 8 is supported. See [Overview of .NET 8 support for SOEs and SOIs](../net-8-support/).

## User interfaces in Portal Home app to enable/disable extensions, reorder the SOIs

Starting from ArcGIS Enterprise 11.3, support is added for enabling/disabling SOEs and SOIs using Portal Home app. Enabled SOIs reordering is possible using Portal Home app. See [Enable Extensions](../enable-extensions-net) for more information.

## Important API changes in 11.3

Added new methods in IImageServerEx interface.
* [PhysicalOutputDirectory](https://developers.arcgis.com/enterprise-sdk/api-reference/net/IImageServerEx/#iimageserverexphysicaloutputdirectory-property)
* [PhysicalCacheDirectory](https://developers.arcgis.com/enterprise-sdk/api-reference/net/IImageServerEx/#iimageserverexphysicalcachedirectory-property)
* [VirtualOutputDirectory](https://developers.arcgis.com/enterprise-sdk/api-reference/net/IImageServerEx/#iimageserverexvirtualoutputdirectory-property)

Added two new interfaces.
* [IServerEnvironmentEx](https://developers.arcgis.com/enterprise-sdk/api-reference/net/IServerEnvironmentEx/) 
* [IServerDirectoryInfo2](https://developers.arcgis.com/enterprise-sdk/api-reference/net/IServerDirectoryInfo2/)
