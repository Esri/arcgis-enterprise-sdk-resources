# Samples

<HeroButton href="https://github.com/ArcGIS/arcgis-enterprise-sdk-resources" newWindow>
  View Samples on GitHub
</HeroButton>

This section provides instructions on how to test ArcGIS Enterprise SDK SOE and SOI samples for both .NET and Java.
To access the source code of these samples, you can visit:

- ArcGIS Enterprise SDK installation location
  - .NET sample code: `C:\Program Files\ArcGIS\EnterpriseSDK\Samples\DotNet`
  - Java sample code: `C:\Program Files\ArcGIS\EnterpriseSDK\Samples\Java`
  - Sample data: `C:\Program Files\ArcGIS\EnterpriseSDK\Samples\EnterpriseSDKSampleData.zip`
- Github repository [arcgis-enterprise-sdk-resources](https://github.com/ArcGIS/arcgis-enterprise-sdk-resources)
  - You can **_[clone or download](https://help.github.com/en/articles/cloning-a-repository)_** this repository for the latest sample code.
