# What's new in 11.2

## Support for 64 bit OID usage in MapServer

Starting with ArcGIS Enterprise SDK 11.2, Query/Find/Identify/RelatedRecords methods to support 64-bit properties. 
For example, in IMapServerEx interface additional methods are added such as [QueryRasterValueEx](https://developers.arcgis.com/enterprise-sdk/api-reference/net/IMapServerEx/#imapserverexqueryrastervalueex-method), [QueryRowCountEx](https://developers.arcgis.com/enterprise-sdk/api-reference/net/IMapServerEx/#imapserverexqueryrowcountex-method), [QueryRowIDsEx](https://developers.arcgis.com/enterprise-sdk/api-reference/net/IMapServerEx/#imapserverexqueryrowidsex-method) etc.
